# Prompt 自动优化系统 — 使用说明

## 一、系统简介

本系统是一个**基于遗传算法（GA）的智能 Prompt 迭代与评估平台**。用户通过 Web 界面提交文本和优化需求，系统自动完成 Prompt 的生成、评分、遗传迭代（共 10 代），最终输出最优 Prompt 及可视化分析报告。

---

## 二、启动项目

### 环境要求

- Python 3.10+
- Django 5.2+
- matplotlib

### 启动命令

```bash
# 进入项目根目录
cd Prompt_Optimization_Based_on_Genetic_Algorithm

# 安装依赖
pip install django matplotlib

# 启动开发服务器
python manage.py runserver
```

启动成功后，在浏览器访问：

```
http://127.0.0.1:8000/
```

---

## 三、页面说明

系统包含两个核心页面：

| 页面 | URL | 功能 |
|------|-----|------|
| 优化器主页 | `/` | 填写任务参数、提交优化、查看实时日志 |
| 结果展示页 | `/results/` | 查看最优 Prompt、适应度曲线、分数分布 |

## 四、使用流程

### 第 1 步：打开优化器主页

浏览器访问 `http://127.0.0.1:8000/`，进入 **Prompt 优化系统** 主界面。

### 第 2 步：填写任务配置

页面中有一张 **任务配置** 表单，包含以下字段：

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| **需求描述** | ✅ 必填 | 描述您希望优化的目标和需求，例如："对以下文章生成一段精炼的摘要" |
| **文本内容** | ✅ 必填 | 输入需要处理的原始文本内容（如一篇文章、一段文档等） |
| **文本标题** | 选填 | 为文本添加标题，帮助系统更好地理解上下文 |
| **作者** | 选填 | 输入文本作者名称 |

#### 填写示例

- **需求描述**：`请对以下文章生成一段 200 字以内的中文摘要，要求涵盖核心观点`
- **文本内容**：`（粘贴您的原始文章内容）`
- **文本标题**：`人工智能的发展与挑战`
- **作者**：`张三`

### 第 3 步：提交并观察实时日志

点击页面底部的 **「开始优化」** 按钮后：

1. 按钮变为 **"优化中…"** 状态（带旋转动画），此时不可重复提交。
2. 页面下方自动弹出 **运行日志** 面板，实时显示后端进度，典型日志流如下：

   ```
   ▸ 开始进行优化步骤
   ▸ 已经生成相关提示词以及文本字典
   ▸ 提示词提取完成
   ▸ 已经完成初始评分 形成种群
   ▸ 已经完成第1代繁殖
   ▸ 已经完成第2代繁殖
   ▸ ...
   ▸ 已经完成第10代繁殖
   ✓ 优化完成，正在跳转到结果页…
   ```

3. 全部 10 代迭代完成后，页面自动跳转到 **结果展示页**。

> **注意**：整个优化过程可能需要数分钟，请耐心等待，切勿关闭页面。若网络中断，日志面板会显示红色错误提示，按钮恢复为"重新开始"，可再次尝试。

### 第 4 步：查看优化结果

跳转到结果页后，您将看到三个核心信息卡片：

#### 4.1 最优 Prompt

- 显示遗传算法终代（第 11 代）中得分最高的 Prompt 文本。
- 左上角的 **Score** 徽章展示该 Prompt 的最终评分。

#### 4.2 适应度演化图

- 折线图展示 11 代种群的 **最高分（Max Score）** 与 **平均分（Average Score）** 变化趋势。
- 用于判断算法是否有效收敛。

#### 4.3 分数分布箱线图

- 箱线图对比第 1 代、第 6 代、第 11 代种群的分数分布。
- 用于观察种群整体质量的提升情况。

### 第 5 步：开始新的优化

在结果页的底部，点击 **「开始新的优化」** 按钮，或点击页面顶部的 **「返回优化页面」** 链接，即可回到主页开始新一轮优化。

## 五、常见问题

### Q1：页面打开后一片空白？
确认 Django 服务已启动且访问地址正确（`http://127.0.0.1:8000/`）。

### Q2：点击"开始优化"后没有日志输出？
- 检查后端终端是否有报错信息。
- 确认 LLM API 密钥和网络连接正常（系统需要调用大模型接口）。

### Q3：结果页图片无法加载？
确认 `static/optimizer/` 目录存在且 Django 静态文件配置正确。图片在优化完成后自动生成到该目录。

### Q4：优化时间过长？
遗传算法默认迭代 10 代，每代涉及多次 LLM 调用。总耗时取决于模型 API 的响应速度，请耐心等待。
